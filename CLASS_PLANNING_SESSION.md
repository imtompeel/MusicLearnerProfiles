# Class Planning Session

## Overview

The "Class Planning Session" is a new interactive session designed to help classes decide what creative projects to do this year. It combines multiple choice questions, word clouds, and open-ended responses to gather input from all students and inform planning decisions.

## Features

### 🎯 **Question Types**

1. **Multiple Choice Questions**
   - Traditional 4-option questions with emojis
   - Students can select multiple answers
   - Real-time response tracking

2. **Word Cloud Questions**
   - Students type individual words
   - Creates visual word collections
   - Great for gathering interests and ideas

3. **Open-Ended Questions**
   - Free-form text responses
   - Character limit (500 characters)
   - Perfect for detailed project ideas

### 📋 **Planning Questions**

1. **Warm-Up: What is creativity?**
   - Multiple choice question about creativity definition
   - Sets the tone for the session

2. **What do you enjoy creating most?**
   - Word cloud question
   - Collects class interests (drawing, music, stories, dance, etc.)

3. **What inspires you?**
   - Word cloud question
   - Gathers inspiration sources (nature, friends, YouTube, colours, music, etc.)

4. **Choose a Creative Superpower!**
   - Multiple choice question
   - Fun way to explore different creative areas

5. **Which tool would you grab first?**
   - Multiple choice with visual options
   - Paintbrush, Guitar, Laptop, Camera

6. **How do you like to share your creativity?**
   - Multiple choice question
   - On stage, Online, In gallery/classroom, With friends/family

7. **Brainstorm Future Projects**
   - Open-ended question
   - Students share project ideas for the year

## How to Use

### For Teachers

1. **Start Planning Session**
   - Select your class
   - Choose "Class Planning Session"
   - Click "Start Planning Session"

2. **Share Student Join URL**
   - Copy the student join URL
   - Share with students on iPads/mobile devices

3. **Guide the Session**
   - Each question has a 60-second timer
   - Advance through questions at your pace
   - View real-time responses from students

4. **Collect Planning Data**
   - All responses are saved automatically
   - Export data for planning meetings
   - Use insights to plan the year's creative projects

### For Students

1. **Join the Session**
   - Enter the session code
   - Type your name to join

2. **Answer Questions**
   - **Multiple Choice**: Tap to select answers
   - **Word Cloud**: Type words and press Enter
   - **Open-Ended**: Write detailed responses

3. **See Your Responses**
   - View your submitted answers
   - See how your ideas compare with classmates

## Question Types in Detail

### Multiple Choice Questions
- **Format**: 4 options with emojis
- **Interaction**: Tap to select/deselect
- **Use Case**: Quick preferences and choices
- **Example**: "What does creativity mean to you?"

### Word Cloud Questions
- **Format**: Text input with word tags
- **Interaction**: Type word, press Enter to add
- **Use Case**: Gathering interests and inspiration
- **Example**: "What do you love making or doing?"

### Open-Ended Questions
- **Format**: Large text area
- **Interaction**: Type detailed responses
- **Use Case**: Project ideas and detailed feedback
- **Example**: "What creative project would you love to try this year?"

## Data Collection

### What's Collected
- Student names and session codes
- All question responses
- Timestamps and participation data
- Class and SoI level information

### How to Use the Data
1. **Export Responses**: Download CSV with all responses
2. **Analyze Trends**: Look for common interests and themes
3. **Plan Projects**: Use student input to design activities
4. **Share Insights**: Present findings to colleagues/parents

## Tips for Success

### Session Management
- Allow enough time for each question
- Encourage participation from all students
- Discuss interesting responses as they come in

### Data Analysis
- Look for patterns and trends
- Consider minority interests too
- Balance popular choices with new experiences

### Follow-Up
- Share planning decisions with students
- Reference their input when starting projects
- Ask for feedback on implemented activities

The Class Planning Session transforms the traditional teacher-led planning process into an engaging, student-centered experience that builds excitement and investment in the year's creative activities!

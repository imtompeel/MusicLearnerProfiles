# My Creativity Session

## Overview

The "My Creativity" session is a new Kahoot-style interactive session designed to explore students' creative musical preferences and approaches. It complements the existing "Music Who Are You?" and "Freesound Session" by focusing specifically on creative expression and musical innovation.

## Features

### 🎨 Kahoot-Style Participation
- **Session Codes**: Students join using unique 6-character session codes
- **Real-time Participation**: Multiple students can join simultaneously using iPads or mobile devices
- **Live Results**: Teachers can see responses in real-time during questions
- **Timer-based Questions**: Each question has a 30-second timer for engagement

### 📱 Multi-Device Support
- **Teacher Interface**: Full control panel on main device
- **Student Interface**: Optimised for tablets and mobile devices
- **Responsive Design**: Works seamlessly across different screen sizes
- **Touch-Friendly**: Large buttons and intuitive navigation

### 🎵 Creative Focus Areas
The session explores various aspects of musical creativity:

1. **Inspiration Sources**: What inspires students to create music
2. **Expression Methods**: How students prefer to express themselves musically
3. **Creative Processes**: What makes students feel creative
4. **Pattern Recognition**: Musical patterns and structures (R2.1+)
5. **Compositional Techniques**: Advanced creative methods (R4.1+)
6. **Cultural Integration**: How culture influences creativity (R6)

## Class Level Adaptations

### R1.1 - Sound Awareness
- Focus on basic creative expression
- Simple multiple choice questions
- Emphasis on sensory and emotional responses

### R2.1 - Pattern Recognition
- Introduction to musical patterns
- Questions about rhythm and structure
- Basic compositional concepts

### R3.1 - Musical Phrases
- Development of musical ideas
- Questions about musical conversations
- Storytelling through music

### R4.1 - Musical Motifs
- Advanced compositional techniques
- Questions about musical unity and variety
- Structural development concepts

### R5.1 - Complete Pieces
- Full composition approaches
- Performance and refinement questions
- Complete musical works

### R6 - Cultural Understanding
- Cultural elements in music
- Questions about identity and values
- Cross-cultural musical expression

## How to Use

### For Teachers

1. **Select Class and Session**
   - Choose your class from the dropdown
   - Select "My Creativity" from the session options

2. **Start Session**
   - A unique session code will be generated
   - Share this code with students
   - Click "Start Session" when ready

3. **Monitor Participation**
   - Watch real-time responses from students
   - See participant count and engagement
   - Use "Simulate Participant Join" for testing

4. **View Results**
   - See answer distributions after each question
   - Navigate through questions
   - End session to save all data

### For Students

1. **Join Session**
   - Open the app on iPad/mobile device
   - Enter the session code provided by teacher
   - Enter your name to join

2. **Answer Questions**
   - Read each question carefully
   - Select multiple answers if desired
   - Watch the timer countdown

3. **See Results**
   - View how your answers compare with others
   - Participate in class discussions about creativity

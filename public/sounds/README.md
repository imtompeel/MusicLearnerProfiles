# Sound Files for R1.1 Students

This folder should contain the following audio files for the "What is your favorite sound?" question:

## Required Files:

1. **nature-sounds.mp3** - Nature sounds (birds, water, wind, etc.)
2. **musical-sounds.mp3** - Musical instruments and melodies
3. **animal-sounds.mp3** - Animal noises (dogs, cats, farm animals, etc.)
4. **everyday-sounds.mp3** - Common everyday sounds (cars, doors, footsteps, etc.)

## File Requirements:

- **Format**: MP3 (recommended) or WAV
- **Duration**: 10-30 seconds each
- **Quality**: Clear, not too loud, appropriate for young children
- **Content**: Should be representative of the category and engaging for R1.1 students

## Usage:

These sound files are automatically played when R1.1 students click the "🔊 Play" button on each preference card. The system will look for these exact filenames in this folder.

## Pattern Matching Sounds (R3.1 Level)

### Question 2 - Animal Sounds:
- `dog_bark.wav` - Dog barking sound
- `cat_meow.wav` - Cat meowing sound
- `bird_chirp.wav` - Bird chirping sound
- `cow_moo.wav` - Cow mooing sound

### Question 3 - Rhythm Patterns:
- `clock_tick.wav` - Clock ticking sound
- `heartbeat.wav` - Heartbeat sound
- `drum_beat.wav` - Drum beat sound
- `rain_drops.wav` - Rain drops sound

## Pattern Matching Requirements

For pattern matching questions, sounds should be:
- Clear and easily distinguishable
- 2-5 seconds in duration
- Consistent volume levels
- High quality recordings

## Note:

If any sound files are missing, the system will show an error message but will continue to function normally. Students can still select their preferences without hearing the sounds.

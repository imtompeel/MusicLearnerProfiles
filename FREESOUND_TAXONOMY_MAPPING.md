# Freesound Broad Sound Taxonomy – App Reference

This reference maps our common labels to Freesound's Broad Sound Taxonomy categories/subcategories and suggests include/exclude tags to improve search accuracy.


The following table provides a more detailed listing of all the available categories and subcategories including descriptions and examples.

| Category           | Subcategory                         | ID       | Description                                                                                     | Examples                                                                       |
| ------------------ | ----------------------------------- | -------- | ----------------------------------------------------------------------------------------------- | ------------------------------------------------------------------------------ |
| Music              |                                     |          | Music excerpts, including melodies, singing, loops, fillers, drones and short musical snippets. |                                                                                |
| Music              | Solo percussion                     | m-sp     | Music excerpts with solo percussive instruments.                                                | rhythmic patterns, unpitched percussion excerpts, drum passages                |
| Music              | Solo instrument                     | m-si     | Music excerpts with only one instrument, excluding percussion.                                  | isolated melody or sequence of notes, solo singing, chords from one instrument |
| Music              | Multiple instruments                | m-m      | Music excerpts with more than one instrument.                                                   | orchestra, band, dueto, multi-instrument composition, multi-instrument loops   |
| Music              | Other                               | m-other  | Music that doesn't belong to any of the above categories.                                       |                                                                                |
| Instrument samples |                                     |          | Single notes from musical instruments, various versions of the same note, and scales.           |                                                                                |
| Instrument samples | Percussion                          | is-p     | Instrument samples that are percussive (idiophones or membraphones).                            | drum, snare, gong, bell, xylophone                                             |
| Instrument samples | String                              | is-s     | Instrument samples that belong to the string instrument family.                                 | guitar, violin, harp, mandolin                                                 |
| Instrument samples | Wind                                | is-w     | Instrument samples that belong to the wind instrument family (aerophones).                      | woodwind, brass, flute, trumpet, saxophone                                     |
| Instrument samples | Piano / Keyboard instruments        | is-k     | Instrument samples of piano or other keyboard instruments, not synthesized.                     | piano, harspichord, organ                                                      |
| Instrument samples | Synths / Electronic                 | is-e     | Instrument samples synthesized or produced by electronic means.                                 | synthesizers (analog/digital), synthesized or electronic samples               |
| Instrument samples | Other                               | is-other | Instrument samples that don't belong to any of the above categories.                            |                                                                                |
| Speech             |                                     |          | Sounds where human voice is prominent.                                                          |                                                                                |
| Speech             | Solo speech                         | sp-s     | Recording of a single voice speaking, excluding singing.                                        | talking, script reading                                                        |
| Speech             | Conversation / Crowd                | sp-c     | Several people talking, having a conversation or dialogue.                                      | playground, people talking in public places, conversations                     |
| Speech             | Processed / Synthetic               | sp-p     | Voice(s) from an indirect source (e.g. radio), heavily processed, or synthesized.               | phone/radio recordings, robotic voice, text-to-speech                          |
| Speech             | Other                               | sp-other | Voice-predominant recordings that don't belong to any of the above categories.                  |                                                                                |
| Sound effects      |                                     |          | Isolated sound effects or sound events, each happening one at a time.                           |                                                                                |
| Sound effects      | Objects / House appliances          | fx-o     | Everyday objects, inside the home or smaller in size.                                           | daily or small objects, tools, weapons, clothes, iron                          |
| Sound effects      | Vehicles                            | fx-v     | Sounds produced from a vehicle.                                                                 | car passing by, car screeching, wiper, car brake, bike, plane, ship            |
| Sound effects      | Other mechanisms, engines, machines | fx-m     | Machine-like sounds, except vehicles and small house electric devices.                          | drill, lawn mower, gear, electric chainsaw                                     |
| Sound effects      | Human sounds and actions            | fx-h     | Sounds from the human body, excluding speech.                                                   | breath, hearthbeat, sneeze, crying, walking, clapping                          |
| Sound effects      | Animals                             | fx-a     | Animal vocalizations or sounds.                                                                 | cat, insect, sheep, animal walk, growl, purr                                   |
| Sound effects      | Natural elements and explosions     | fx-n     | Sound events occurring by natural processes.                                                    | wind, fire, ice, rocks, water, stones, explosions                              |
| Sound effects      | Experimental                        | fx-ex    | Experimental sounds or heavily processed audio recordings.                                      | reversed sounds, weird effects, unusual recording techniques or processing     |
| Sound effects      | Electronic / Design                 | fx-el    | Sound effects that are computer-made or designed for user interfaces or animations.             | sci-fi, laser, whoosh, boink, cartoon, ui, sound alerts or notifications       |
| Sound effects      | Other                               | fx-other | Sound effects that don't belong to any of the above categories.                                 |                                                                                |
| Soundscapes        |                                     |          | Ambiances, field-recordings with multiple events and sound environments.                        |                                                                                |
| Soundscapes        | Nature                              | ss-n     | Soundscapes from natural habitats.                                                              | forest ambiance, seaside, river with surrounding nature, farmland              |
| Soundscapes        | Indoors                             | ss-i     | Soundscapes from closed or indoor spaces.                                                       | closed room ambience, room tone, office, factory, bar                          |
| Soundscapes        | Urban                               | ss-u     | Soundscapes from cityscapes or outdoor places with human intervention.                          | city ambience, outside airport, busy road                                      |
| Soundscapes        | Synthetic / Artificial              | ss-s     | Soundscapes that are synthesized or computer-made ambiences.                                    | artificially-created soundscapes, soundscapes from imaginary places            |
| Soundscapes        | Other                               | ss-other | Soundscapes that don't belong to any of the above categories.                                   |                                                                                |
